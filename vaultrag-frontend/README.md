# VaultRAG Frontend

This is the Next.js frontend application for **VaultRAG**.

*   Built on React 19 and Next.js 16+.
*   Utilizes strict `TypeScript` components (`.tsx`).
*   Configured with Tailwind CSS and custom glassmorphism styles in `app/globals.css`.
*   Includes client-side text-to-speech visualization and persistent storage.

## Development

Run `npm install` and then `npm run dev` to access the UI at `localhost:3000`. 
**Note:** Ensure the Python backend (`uvicorn main:app --reload`) is running.
